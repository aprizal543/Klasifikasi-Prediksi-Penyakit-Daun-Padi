# Klasifikasi-Prediksi-Penyakit-Daun-Padi
Proyek ini dibuat untuk mengklasifikasikan penyakit pada daun padi. Dan dataset yang saya gunakan dapat diunduh pada link berikut ini: https://universe.roboflow.com/dfinazrmd/deteksi-penyakit-daun-padi-3e5p1.
Adapun model yang saya pakai, yaitu menggunakan Transfer Learning - Mobile Net
Akurasi pada data Train nya mencakup sekitar 95%
Akurasi pada data Validasi nya mencakup sekitar 97%
Dari hasil Akurasi tersebut Model ini sudah cukup baik dalam mengklasifikasikan penyakit pada daun padi.
